# Module Federation for a monorepo built with Vite, React, and React Router

This repository is a demo repository of an implementation of Module Federation for a monorepo built with Vite, React and React Router.

The monorepo contains a host application and a remote application. The host fetches and renders the remote application at runtime. The remote application can be run as a standlone application as well.

## Running the demo

To run the demo, simply execute the following commands from the root of the repo,

```bash
npm i

npm run dev
```
